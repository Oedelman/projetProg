SittR:
Nous proposons une application rapide, directe et simple d’utilisation qui permet de trouver un/une baby-sitter.
Nous avons deux types d’utilisateurs principaux, le premier offre un service de baby-sitting, le second recherche ce même service. 
Les utilisateurs pourront ensuite trouver quelqu'un qui leur correspond grâce à un système de 'matching' qui leur donnera leurs coordonnées
afin de les contacter.

Informations:
Cette application Meteor a été réalisée dans le cadre du cours "Programmation pour Internet II (SP 2017)" donné par Isaac Pante.
Elle a été conçue afin d'être adaptée aux téléphones mobiles.
 
Afin d'installer cette application:
1. Installer Meteor à https://www.meteor.com/install.
2. Démarrer l'invite de commandes.
3. Accéder au dossier de l'application en ajoutant 'cd' avant le chemin du dossier contenant l'application.
4. Lancer l'application en tapant la commande "meteor run".
5. Si vous souhaitez utiliser un émulateur mobile, allez sur https://guide.meteor.com/mobile.html, suivez le tutoriel puis tapez "meteor run android" dans l'invite de commandes.
